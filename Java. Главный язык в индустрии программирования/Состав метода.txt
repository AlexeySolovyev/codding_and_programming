СОСТАВ МЕТОДА

1. Доступность
	public - публичный, доступен для использования в других классах.
	private - приватный, доступен только в текущем классе.
	protected - пакетовый, доступен только внутри своего и других пакетов.

2. Идентификатор
	static - статичный, принадлежащий типу, а не объекту.
	final - нередактируемый.

3. Возвратность
	void - любой или отсутствует.
	Указывает, какой тип данных возвращает метод.
	Если тип указан, то обязателен в конце кода параметр return.
