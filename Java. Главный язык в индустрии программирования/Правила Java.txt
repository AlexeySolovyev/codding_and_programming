СТРУКТУРА ПРОЕКТА

Название проекта отражает его суть, название создаваемой программы.
По стандарту название пакета: страна.компания.название_проекта - ru.pavetranquil.project
Название класса отражает то, что он делает.


Project
	src
		Package
			Class.java
	out
		Class.class



СТРУКТУРА КЛАССА
Название метода отражает то, что он делает.


package название_пакета;
import название_класса;

public class НазваниеКласса {
	public static void main(String[] args) {
		код;
		код;
		...
	}
	
	public void названиеМетода() {
		код;
		код;
		...
	}
}
